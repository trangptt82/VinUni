# Grading Rubric: Day 10 Lab

## Total Points: 100

### 1. ETL Automation Script (40 Points)
- **Functional (20 pts)**: Script runs without crashing.
- **Validation (10 pts)**: Successfully drops invalid records (e.g., negative price).
- **Transformation (10 pts)**: Category normalization and discount calculation are correct.

### 2. Data Observability & Logging (20 Points)
- **Status Reporting (10 pts)**: Script prints how many records were processed vs dropped.
- **Timestamps (10 pts)**: Processed records include metadata (e.g., `processed_at`).

### 3. Stress Test Experiment (20 Points)
- **Completion (10 pts)**: Table filled out with Agent responses for Clean vs Garbage data.
- **Critical Thinking (10 pts)**: Solid observation on *why* the agent failed on garbage data.

### 4. GitHub Best Practices (20 Points)
- **Structure (10 pts)**: Files are organized correctly in the repo.
- **README (10 pts)**: Clear instructions on how to run the pipeline.
