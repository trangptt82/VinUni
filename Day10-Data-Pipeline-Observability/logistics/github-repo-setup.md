# GitHub Repository Setup (Student Guide)

To manage your work for Day 10, please follow these steps to set up your repository.

## 1. Create a New Repository
- Name: `AI20K-XXXX-Day10` (Replace `XXXX` with your Student ID).
- Visibility: **Public**.
- Initialize with a `README.md`.

## 2. Structure Your Repository
Please organize your files as follows:
```text
/
├── README.md                 (Introduction & How to run)
├── solution.py               (Your ETL script)
├── processed_data.csv        (The output of your script)
├── experiment_report.md      (Your findings from the Agent lab)
└── logs/                     (Optional: Any log files generated)
```

## 3. Pushing Changes
```bash
git add .
git commit -m "feat: complete Day 10 Lab 10"
git push origin main
```

## 4. Submission
Paste your repository URL into the LMS for grading. **Double check that the repo is Public!**
